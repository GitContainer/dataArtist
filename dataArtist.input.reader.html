class dataArtist.input.reader.csv.CsvPlotReader(*args, **kwargs)[source]¶

Bases: dataArtist.input.reader._ReaderBase.ReaderBase

Readout values from a plain-text saved in file. The format has to be similar to the following example:

value00 value01 value02
value10 value11 value12
...

Where the values of all dimensions are written in one line, separated by one or more signs like spaces ‘ ‘ or tab ‘ ‘.

Large files (larger that RAM) can be read through choosing a buffer size.
